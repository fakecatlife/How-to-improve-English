# Ways to improve English:

1. **Reading:** Read English content such as newspaper, blog, books as much as you can.
2. **Writing:** Write down what you have been read by looking at textbook, newspaper.
3. **Listening:** Listening is most crucial part in learning any language. You can watch English movies, TV shows, videos.
4. **Speaking:** You can try little bit speaking with friends, teachers. If you're not confident at first glance then avoid this for 2-3 months.
5. **2 words, 2 Idioms:** Write 2 new English words, 2 new Idioms in notebook daily.
6. **Entertainment for learning:** Pick any movie or TV show, start watching it, enable Subtitle, pause video at every dialogue, repeat it loud.

Keep it consistent, you will be mastered English in a year. If you trust me then employee this 6 ways. I have been doing this since long time and my English fluency is good now.

# English
## Ways to improve English:

1. **Reading📖:** Read English content such as newspaper, blog, books as much as you can.
2. **Writing✍️:** Write down what you have been read by looking at textbook, newspaper.
3. **Listening👂:** Listening is most crucial part in learning any language. You can watch English movies, TV shows, videos.
4. **Speaking📢:** You can try little bit speaking with friends, teachers. If you're not confident at first glance then avoid this for 2-3 months.
5. **2 words, 2 Idioms✍️:** Write 2 new English words, 2 new Idioms in notebook daily.
6. **Entertainment for learning🍿:** Pick any movie or TV show, start watching it, enable Subtitle, pause video at every dialogue, repeat it loud.

> Keep it consistent, you will be mastered English in a year. If you trust me then employee this 6 ways. I have been doing this since long time and my English fluency is good now.

# मराठी
इंग्रजी सुधरायचे पद्धीती:

1. **वाचणे:** इंग्रजी वर्तमाणपत्र वाचा, पुस्तके वाचा.
2. **लिहिने:** जे तूम्ही वाचता तेच बघून लिहून काढा.
3. **ऐकणे:** ऐकणे खूप महत्वाचा भाग आहे, म्हणून इंग्रजी चित्रपट, इऔग्रजी YouTube video, आणि इंग्रजी मालीका बघा.
4. **बोलने:** इंग्रजी मध्ये बोलायचा try करा.
5. **2 शब्द, 2 म्हणी:** रोज 2 इंग्रजीचे नवीन शब्द वहीत लिहा आणि रोज 2 म्हणी देखील लिहा.
6. **मनोरंजन इंग्रजीसाठी:** एक चित्रपट निवडा, subtitle चालू करा, जे वाक्य कलाकार म्हणतात ते तूम्ही देखील त्यांच्यासोबत म्हणा.

> या गोष्टी रोज करा आणि 6-7 महिन्यात तूम्हाला result मिळेल.
